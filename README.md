# Olist E-commerce Analytics (dbt + Snowflake)

## Overview
This project builds a modern ELT pipeline using dbt on top of Snowflake.

## Architecture
Raw → Staging → Intermediate → Marts

## Key Features
- Incremental models
- Data quality tests
- Modular data modeling

## Data Source
Olist E-commerce dataset

## How to run
dbt run
dbt test
